## Detailed Description
```
TCP Contract Processor is an intelligent document processing system that extracts and standardizes data from Time Charter Party (TCP) maritime contracts. Built with Claude AI, it features both a modern Streamlit web interface and a full-featured command-line interface for flexibility and automation.
```
